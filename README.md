# lapis
A lightweight DE for Wayland


### Dependencies 

Lapis relies on a few dependencies to be able to run 

- gtk4
- gtk4-layer-shell
- wayland 
- a window manager

> [!NOTE]
> Lapis has only been tested on Hyprland and so behaviour might not be optimal on your window manager of choice
